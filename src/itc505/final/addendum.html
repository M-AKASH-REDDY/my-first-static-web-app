Addendum: Detailed Description of LIGHTS OUT Game Development

Objective:

The Lights Out game is an interactive web-based puzzle designed to challenge players in turning off all the lights on the game board. Developed using HTML, CSS, and JavaScript, the game provides an engaging and visually appealing experience.

HTML Organization:

  1. Game Title: To highlight the identity of the game, the title is shown beside the element in a prominent manner.
  2. Game Board: Each square on the game board, which is a grid inside the class board, is defined as a child.
  3. Buttons and Metrics: A new game is started by clicking the "New Game" button. Elements are used to represent metrics like a timer and the quantity of moves.

Logic in JavaScript:

  1. Moves and Move Limit: To add a challenge, the moves variable tracks the player's movements and has a defined MAX_MOVES limit. The player can see the move count.
  2. Timer Functionality: A timer that counts down in minutes and seconds is updated every second.
  3. Square Toggling: Clicking on a square starts a chain reaction that toggles both that square's state and the states of the squares next to it.
  4. Game State Checking: When every square is disabled, the game looks for a win condition (isWin). If the move limit is reached, a loss condition is set off.
  5. Warnings on Win/Loss: Congratulations or the game is finished, together with the amount of moves and duration, are shown.

Game Plot:

  1. Initialization: The game starts a new game and sets the moves and timer to zero at the same moment.
  2. Square Interaction: Clicking on a square causes the square and the squares next to it to toggle states.
  3. Game State Check: The game checks to see if it has won or lost after every move.
  4. End Game Alerts: The player receives feedback via alerts that appear after a win or loss.

Conclusion:

Lights Out is a puzzle game that combines logical implementation, interactive design, and visual appeal to create a fun and challenging puzzle experience. To create a satisfying gaming experience, players are advised to plan ahead and effectively switch off every light within the allotted move limit.
